Portfolio 1

The aim of this first portfolio piece is to document my search process of relevant data for my topic. I have consistent interest in the topic of traumatic brain injuries and mental health issues. I hope to find data that can speak to this topic; however, this portfolio is to introduce my research process and decide on a data set for the rest of my inquires.

I will begin by loading the tidyverse packages that can assist with further analysis of any data set I find.

library(tidyverse) 
library(tidyr)

I started my search with the “Harvard Dataverse” database. In the search bar I began with the prompt “(traumatic brain injury) AND (mental health)”. This search prompt yielded 2,040 results. An initial difficulty I stumbled across is that many of the data sets are restricted due to containing unique patient information. In order to carry on my search I had to filter my results to only include data sets that are set to public, and in doing so limited the results to 420.

I initially explored a data set that was interesting to me that observes the relationship between childhood trauma and mentalization capacities and dissociative experiences. I was able to successfully open this data set in SPSS and observe the variables measured more closely. This data set consists of 101 participants and 448 variables that would be extremely helpful in observing a variety of inquires. However, I still want to continue my search.

Next, I examined a different database at “data.gov”, and broadened my search to “concussions and mental health”. So far, I came across a public data set for “Hospital Encounters for Behavioral Health”. I was able to open a .csv file that contains variables such as sex, race, and visit reasons. After looking at this extensive data, I have decided to explore more in depth the summary of data of visits for behavioral health related issues.

I have downloaded the data set in SPSS, and imported a .sav file in Rstudio for the purposes of examining the data more closely.
